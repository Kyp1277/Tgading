# Folder Foto Struktur Organisasi

Upload foto formal atau foto khusus untuk bagan **Struktur Organisasi** ke dalam folder ini.

## Ketentuan File
- Tempatkan file foto Anda di sini dengan nama yang sesuai dengan konfigurasi pada file `src/data/anggotaData.js`.
- Contoh default yang telah disiapkan:
  - Hamdi Gustiawan: `hamdi.jpg`
  - Lisa Nopitasari: `lisa.jpg`
  - Nur Annisa: `nur_annisa.jpg`
  - Tasya Salsabilla: `tasya.jpg`
  - Rifky Naufal Athaya: `rifky.jpg`
  - Wardi Siregar: `wardi.jpg`
  - Walfajri: `walfajri.jpg`
  - Nadira Solekha: `nadira.jpg`
  - Anisa Febrianti: `anisa_febrianti.jpg`
  - Namira Sahida: `namira.jpg`
  - Dian Purwati: `dian.jpg`

*Catatan: Jika Anda menggunakan format gambar lain seperti `.png` atau `.webp`, silakan sesuaikan nama filenya di file konfigurasi `src/data/anggotaData.js`.*
